# CareQuest Manifesto

AI-native workforce infrastructure for neurodivergence care.

## Core Thesis

The biggest bottleneck in neurodivergence care is not demand.

It is trained human supply.

The world does not have enough job-ready care professionals for autism, ADHD, behavioral support, inclusive education, and developmental care.

Most companies are building:
- therapy delivery apps
- telehealth platforms
- clinic software
- wellness tools

CareQuest is exploring the infrastructure required to create, train, verify, and deploy the next generation of neurodivergence care workers.

## Starting Wedge

Help ABAT/RBT training providers increase learner completion and certification readiness.

## Why This Matters

Care work is not only a clinical problem.

It is a workforce infrastructure problem.

If we can train more people better, faster, and with measurable competency, we can unlock access to care globally.

## What We Are Building Toward

- gamified certification pathways
- AI behavioral simulations
- competency scoring
- AI supervision
- workforce graphs
- readiness prediction
- placement infrastructure

## Long-Term Vision

A global operating system for emotionally intelligent human care labor.
